Priority+ scroll with Dragscroll microlibrary used on modal tabs navigation

Minimize window for better view of scroll navigation

This supports only menu-navigation without dropdowns.

HTML:
<ul class="nav nav-tabs dragscroll" role="tablist">
  <li role="presentation" class="active"><a href="#home" aria-controls="home" role="tab" data-toggle="tab">Home</a></li>
  <li role="presentation"><a href="#profile" aria-controls="profile" role="tab" data-toggle="tab">Profile</a></li>
  <li role="presentation"><a href="#home2" aria-controls="home2" role="tab" data-toggle="tab">Home</a></li>
  <li role="presentation"><a href="#profile2" aria-controls="profile2" role="tab" data-toggle="tab">Profile</a></li>
  <li role="presentation"><a href="#messages" aria-controls="messages" role="tab" data-toggle="tab">Messages</a></li>
</ul>
CSS:
@media screen and (max-width: 991px){
  .nav-tabs > li					{float: none; display: inline-block;} 
  .nav-tabs						{white-space: nowrap;}	
  .nav-tabs.dragscroll				{overflow-x: auto; overflow-y: hidden;}
  .nav-tabs.dragscroll::-webkit-scrollbar 		{display: none; -ms-overflow-style: none; width: 0; height:0}
}
